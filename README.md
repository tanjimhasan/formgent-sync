## Installation

1. Clone Github Repository

   ```sh
   git clone https:://github.com/sovware/formgent.git
   ```

2. Install composer dependencies
   ```sh
   composer setup
   ```

3. Install npm dependencies (Node v20.10.x)
   ```sh
   npm install
   ```

4. Compile Scripts
   ```sh
   npm start
   ```

4. Build plugin
   ```sh
   npm run build
   ```
